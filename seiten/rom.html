ROM

Beschreinung

ROM (Read Only Memory) ist ein nichtflüchiges Speicher, welches meistens zur Speicherung von unveränderbaren Daten verwendet wird.

Verwendung

ROM (Read Only Memory) ist ein nichtflüchiges Speicher, welches meistens zur Speicherung von unveränderbaren Daten verwendet wird. Auf einem Mikrocontroller wird diese zum Beispiel als sogenannten Mask ROM verwendet. Mask ROM-Chips werden bei der Herstellung mit Daten beschrieben, welche nicht verändert werden können. Ein Anwendungspeispiel für solche Chips ist die Speicherung von Firmware.
