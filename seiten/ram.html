RAM

Beschreibung

RAM ist eine flüchtige Speichertechnologie, welche unbegrenzt an jeder stelle beschrieben werden kann was in in der Thoerie für jeden Verwendungszweck nutzar macht. Im verleich zu Flash (welcher Blockweise beschieben werden muss), kann RAM an jeder beliebeigen stelle beschreiben und freigegeben werden mithilfe einer Adresse inerhalb des speichers. der Name RAM (Random Access Memory) ist irreführend da dies beispielsweise auch für ROM gilt, grunddessen solle er eigendlich zu RWM(Read Write Memory) umbenannt werden, dieser begriff hat sich jedoch nicht durchgesetzt.

Eigenschaften

Anwendung

Auf Mikrocontrollern wird der RAM zum Beispiel dazu verwendet um analoge und digitale Signale oder Variablen vorrübergehend zu Speichern.
